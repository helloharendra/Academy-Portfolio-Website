- 📖 **About** – Academy mission and vision
- 📚 **Courses** – Display available courses with details
- 💬 **Testimonials** – Student feedback carousel/grid
- ⭐ **Features** – Key highlights and benefits
- 📱 **Fully Responsive** – Works on all devices
- 🎨 **Component-Based Architecture** – Reusable React components
- ⚡ **Fast & Optimized** – Built with Vite

## 🚀 Getting Started

1. **Clone the repo**
   ```bash
   git clone https://github.com/helloharendra/Academy-Portfolio-Website.git
   ```

2. **Navigate to project directory**
   ```bash
   cd Academy-Portfolio-Website
   ```

3. **Install dependencies**
   ```bash
   npm install
   ```

4. **Run development server**
   ```bash
   npm run dev
   ```

5. **Build for production**
   ```bash
   npm run build
   ```
